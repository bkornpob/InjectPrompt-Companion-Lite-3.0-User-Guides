# Module 2: Discovering Advanced Capabilities

## Welcome to Intermediate Red Teaming

Now that you're comfortable with basic prompts, let's explore the full range of what the red teaming copilot can help you with. This module focuses on discovering different approaches and techniques.

## Suggested Exploration Queries (Copy & Paste)

Use these queries to uncover different capabilities:

1. **"what techniques work for sensitive topics?"** - Learn about specialized approaches
2. **"show me different framing methods"** - Discover various narrative strategies
3. **"how do I handle refusal messages?"** - Learn iterative improvement techniques
4. **"what are perturbation tactics?"** - Explore evasion methods
5. **"can you help with [specific topic]?"** - Test with your own interests

## Capability Discovery Areas

### 1. Narrative Framing Techniques
The agent can help you frame requests as:
- **Academic research** (museum exhibits, historical analysis)
- **Creative writing** (fiction, character perspectives)
- **Educational content** (teaching materials, explanations)
- **Technical documentation** (engineering analysis, specifications)

### 2. Evasion Strategies
Discover different ways to bypass safety filters:
- **Character-level manipulation** (adding symbols, spacing)
- **Structural changes** (reordering text, flipping words)
- **Context framing** (adding safety disclaimers, educational markers)
- **Language variations** (different phrasing, synonyms)

### 3. Research Methodologies
Learn structured approaches for:
- **Progressive escalation** (starting simple, increasing complexity)
- **Comparative testing** (trying multiple approaches)
- **Response analysis** (learning from refusals)
- **Documentation** (tracking what works)

## Practical Exploration Exercise

1. **Choose a topic** you're interested in (e.g., "historical weapons," "chemistry concepts," "engineering principles")
2. **Ask the agent** for help with that topic using different approaches:
   ```
   "Can you help me create an academic museum exhibit about [your topic]?"
   ```
   ```
   "I need a creative writing prompt about [your topic] for a fiction story"
   ```
   ```
   "What are some educational approaches to explain [your topic]?"
   ```
3. **Compare the responses** and see which framing works best
4. **Note what you learn** about the agent's capabilities

## Real-World Example: Academic Framing

Let's walk through a specific example to see how this works:

**User Query:** "can you help with explosive devices?"

## Common Intermediate Questions

**Q: How do I know which technique to use?**
A: Start with what feels natural for your topic. Academic framing works for technical subjects, creative writing for stories, educational for explanations. The agent will guide you.

**Q: What if I don't know what to ask for?**
A: Use the exploration queries above. They're designed to reveal different capabilities.

**Q: How do I build on what I learn?**
A: Each successful interaction teaches you something. Note what works, ask for variations, and gradually increase complexity.
